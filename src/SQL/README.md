
### Comandos docker

Subindo conteiner:	  `$ docker-compose up` 
Subindo em backgroud: `$ docker-compose up -d` 
Conteiners rodando:   `$ docker ps`
Conteiners existentes:`$ docker ps -a`
Iniciando conteiner:  `$ docker start <ID>`
